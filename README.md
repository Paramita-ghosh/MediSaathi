# 🩺 MediSaathi — Your Intelligent Medication & Wellness Companion  

---

## 🧭 Project Overview  

**MediSaathi** is an AI-powered health and medication management platform built to help users stay consistent with their prescribed medicines while maintaining wellness insights, community support, and smart notifications.  
It merges **personal healthcare**, **AI guidance**, and **beautiful design** — making medicine tracking simple, interactive, and human-friendly.

---

## ✨ Core Features  

### 👤 Authentication  
- Secure **JWT-based Register & Login**  
- Real-time validation and smooth motion transitions  
- Clean and accessible design with **TailwindCSS + Framer Motion**

### 💊 Medication Tracker  
- Add, update, or delete medicines with dosage and timing  
- Automatic tracking of **taken** and **missed** doses  
- Live adherence chart that updates with every logged dose  

### 📈 Adherence Analytics  
- Visual representation of medicine consistency  
- **Recharts**-based graphs with animation  
- Shows total adherence rate (%), taken vs missed stats  

### 🤖 AI Assistant — “MediBot”  
- Smart chatbot powered by **ChatGroq API**  
- Helps you remember schedules, get dosage help, or ask medical queries  
- Context-aware conversation flow with friendly personality  

### 💬 Community Forum  
- Share posts, tips, and health experiences  
- Like posts and engage with other users  
- Interactive motion effects and card-based layout  

### 📧 Email Notifications  
- Email notifications on adding new medicine dose using **SendGrid** 

### 🧭 About & Design Philosophy  
- Minimal, modern, and soothing health-themed interface  
- Motion-first design using **Framer Motion**  
- Includes fade, slide, and glow animations for smoother transitions
- Subtle pastel gradients for calm medical tone
- Fully responsive for mobile and tablet

---

## 🧱 Tech Stack  

### 🌐 Frontend  
- React.js (Vite)  
- TailwindCSS  
- Framer Motion  
- React Icons  
- Axios  

### ⚙️ Backend  
- Node.js + Express.js  
- MongoDB with Mongoose  
- JWT Authentication  
- SendGrid API (Email Notification)  
- ChatGroq API (AI Assistant)
